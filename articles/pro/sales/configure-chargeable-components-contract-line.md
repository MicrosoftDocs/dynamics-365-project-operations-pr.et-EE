<span data-ttu-id="96f5d-106">Kaasatud komponendid on komponendid, mis kuuluvad:</span><span class="sxs-lookup"><span data-stu-id="96f5d-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="96f5d-113">Mittearveldatav</span><span class="sxs-lookup"><span data-stu-id="96f5d-113">Non-chargeable</span></span>

<span data-ttu-id="96f5d-114">Arveldatavad komponendid saab määratleda tööülesannetes, rollides ja tehingute kategooriates.</span><span class="sxs-lookup"><span data-stu-id="96f5d-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="96f5d-121">Projekti tööülesande värskendamine arveldatavaks või mittearveldatavaks</span><span class="sxs-lookup"><span data-stu-id="96f5d-121">Update a project task as chargeable or non-chargeable</span></span>
