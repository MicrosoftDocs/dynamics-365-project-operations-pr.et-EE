# <a name="bookings-vs-assignments"></a>Reserveeringud ja määramised

_**Rakendub:** Project Operationsi ressurssipõhiste/mitteaktsiapõhiste stsenaariumide korral,  Lihtjuurutamine - tehing fiktiivsele arveldusele_

Broneeringud on ressursside fikseeritud või esialgsed projektile eraldamised. Fikseeritud broneeringud tarbivad ressursivõimsust. Reserveeringud annavad meeskondadele korralduslikud kontseptsioonid, et nad mõistaksid, kuidas ressursid erinevate projektide üleselt rakendatakse. Dynamics 365 Project Operations loeb reserveeringud projekti tasemel kontseptsiooniks. 

Erinevalt broneeringutest on määramised ressursside kohustus projekti ülesannete osas projekti ajakavas. Ressursid võivad olla nimelised või üldised. 

Tavaliselt võrdub ressursi reserveeringute summa ressursi määramiste summaga ühe või mitme ülesande üleselt. Samas Project Operations ei jõusta seda kokkulepet. Vaates **Vastavusseviimine** kuvatakse projektijuhile kohti, kus ressursi broneeringud ja määramised ei ühti.
